## Use

### 第一次拉取代码：
```
git clone https://github.com/Gongzq5/NUS-DSA5101-Project-1.git
```
### 本地写完了上传代码：

```
git add *
git commit -m ""
git push
```

> -m "" 这个引号里简单描述一下这次commit的内容，随意

### 拉取远程仓库里的更新内容：
```
git pull
```
可能需要先 
```
git add *
git commit -m ""
```
> -m "" 这个引号里简单描述一下这次commit的内容，随意
